Somos Editores HTML

Colegio Frances Hidalgo, Profesor José Luis Lobera

Logo del lenguaje HTML

HTML, siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

En este proyecto el alumno programará una página web sencilla, usará las etiquetas HTML para crear marcas básicas, empleará hipervínculos para conectar ideas y fuentes, descubrirá cómo Internet hace que la web sea posible y comprenderá los riesgos y responsabilidades al momento de usar Internet.

Internet

Todos usamos Internet de distintas maneras. A menudo, se utiliza para la comunicación y la colaboración. Por ejemplo, podemos comunicarnos empleando el correo web y trabajar en equipo con herramientas como Documentos de Google.

Internet es una red mundial de computadoras conectadas por cables de cobre, fibras ópticas, redes inalámbricas y satélites. Permite al navegador web de una computadora, como Chrome o Safari, el acceso, la visualización y la interacción con los documentos almacenados en otras computadoras conectadas a ella que se denominan servidores web. Estos documentos son lo que conocemos como páginas web.

Las páginas web se localizan mediante su URL, el cual se compone de diferentes partes que tienen significados específicos.

Páginas web

Una página web, página electrónica, página digital o ciberpágina, es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes, hipervínculos y muchas otras cosas, adaptada para la llamada World Wide Web (WWW), y que puede ser accedida mediante un navegador web.

Cuando visitas una página web, su apariencia (colores, estilo e imágenes) es el resultado del código informático que hay detrás de ella. Tal código se denomina HTML y puedes acceder a él usando la herramienta ver código fuente del navegador.

Etiquetas HTML

Como hemos aprendido, HTML es el lenguaje de marcas con el que se crean las páginas web. Este utiliza etiquetas para definir o delimitar las distintas partes de una página web; con ellas, puedes controlar la apariencia de esta y cómo se organiza su contenido.

Creación de una página web

Editor de texto es un programa informático que permite crear y modificar archivos digitales compuestos únicamente por textos sin formato, conocidos comúnmente como archivos de texto o “texto plano”. El programa lee el archivo e interpreta los bytes leídos según el código de caracteres que usa el editor.
